# Student Grade Predictor

## Project Overview

This project predicts student performance using Machine Learning.

## Dataset Features

- Study Hours
- Attendance Percentage
- Math Score
- Science Score
- English Score
- Final Grade

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm

Linear Regression
Random Forest Classifier

## Features

- Grade Prediction
- Student Performance Analysis
- Correlation Analysis
- Interactive Visualizations

## Results

The model predicts student scores and grades based on academic performance indicators.
